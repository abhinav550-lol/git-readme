Stats Card:
Your README or webpage uses an image URL like:
https://some-site.com/api/github-stats?username=abhinav550
The browser sees it as a normal image and sends a request to that backend.
The backend:
reads the username query param
calls GitHub API
calculates stats
builds a visual layout
returns an image response
The response has headers like:
Content-Type: image/svg+xml or image/png
cache headers so GitHub/README/browser does not re-fetch too often
